LTI Extensions

Introduction

LTI provides a great base for creating a connection between learning tools. These extensions are used to add functionality on top of that connection. These are unofficial and not all consumers and providers will support them.

Extensions
Name Description
Content This extension is used to pass content to the tool consumer from the tool provider.
Result Data Add resultData in addition to resultScore in outcome services

Vendor-Specific Configurations

These aren't LTI extensions, but applications of the extensions in some tools/platforms. These pages show examples and have info for setting up tools with these extensions.

Vendor Name Description
Canvas Navigation Add LTI links as navigation items in courses/accounts/user areas
Canvas WYSIWYG Add content to WYSIWYG editor using the content extension
Canvas Link Selection Add LTI launch urls to modules using the content extension
Canvas Homework Turn in homework assignments using the content extension
