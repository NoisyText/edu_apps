XML Config Builder

Helping you build LTI XML configs, one step at a time.

Introduction

LTI configuration is powerful and flexible, but not necessarily easy to set up. Rather than asking everyone to rebuild configurations from scratch, here's a little tool for quickly building LTI configurations. My advice is to check some boxes and stuff until you figure it out. Most of the options should be clear if you're familiar with LTI and the Canvas extensions.
